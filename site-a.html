Composant auto chargée SITE A

Le but est de permettre à des comporants tiers isolés dans des fichiers d'être automatiquement exécuté en cas de présence dans le HTML.

  • Le composant doit être dans un fichier séparé.
  • Il ne doit pas être nécessaire de faire un appel explicite dans le code.
